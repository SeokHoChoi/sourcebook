# System Design Interview Research Notes

이 폴더는 `대규모 시스템 설계 기초` 트랙의 외부 근거 메모를 모아 두는 곳이다.

현재 단계는 목차 기반 스캐폴딩 단계라서, 장별 본문을 모두 검증한 상태는 아니다. 대신 앞으로 장별 캡처가 들어올 때 바로 확장할 수 있도록 아래 원칙으로 시드를 깔아 둔다.

1. 기술 사실은 공식 문서, 논문, 원저자 자료를 먼저 본다.
2. 빅테크 엔지니어링 글과 GitHub 저장소는 실무 구현 감각을 보강하는 용도로 쓴다.
3. Reddit, Stack Overflow, OKKY 같은 커뮤니티는 자주 생기는 오해와 실무 감각을 잡는 보조 근거로만 쓴다.
4. 프론트엔드에 직접 연결되는 사례를 우선 수집하되, 풀스택 확장 포인트는 버리지 않는다.

현재 초기 목록은 `source-registry.md`에 정리했다.
